@nestjs/config: centraliza e gerencia as configurações da aplicação, essencialmente substituindo o uso direto de process.env com uma abordagem mais organizada e poderosa

@nestjs/jwt: Pacote para gerar e validar tokens JWT (JSON Web Tokens)

@nestjs/passport: Integra estratégias de autenticação do Passport.js (local, jwt, google, facebook, etc.)

@prisma/client: O ORM para interagir com o banco de dados.

bcrypt: Para "hashear" as senhas dos usuários (nunca salve senhas em texto puro!).

class-validator: Valida se os dados que chegam no seu DTO são corretos (ex: se o email é realmente um email).

class-transformer: Transforma objectos simples em instâncias de classes e vice-versa.

@nestjs/mapped-types: Criar variações de DTOs (como o PartialType para atualizações de dados).
